Mount your own files
Add a volume flag to access local data inside RStudio and the terminal:
docker run -d -p 80:80 -p 8787:8787 -p 8080:8080 \ -e PASSWORD=phylo123 --name mp26 \ -v ~/mydata:/home/rstudio/mydata \ mahmoudxyz/mp26-phylo
Your files appear at ~/mydata inside RStudio and the terminal.
